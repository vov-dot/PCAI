
# PCAI


Инструмент для локального запуска ИИ-помощника, который интегрируется с вашими рабочими документами и почтой.  
    
**Настройка Gmail:**  
   
Для активации функций Gmail необходимо выполнить следующие шаги:  

   1. Google Cloud Project: Создайте новый проект в консоли Google Cloud.
   2. API & Services: Включите Gmail API для вашего проекта.
   3. Credentials:
   * Создайте учетные данные OAuth 2.0.
      * Скачайте файл и переименуйте его в credentials.json.
   4. Access: Поместите файл в корневую директорию приложения и предоставьте приложению доступ к вашему Gmail аккаунту при первом запуске.

🛠 Возможности

* Word: Анализ и генерация текстового контента.
* Excel: Автоматизация расчетов и работа с таблицами.
* Gmail: Умная сортировка и написание ответов.
